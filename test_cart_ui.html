Troubleshooting

If cart UI doesn't appear:

  1. Wait 2-3 minutes for CloudFront cache to clear
  2. Hard refresh the page (Ctrl+Shift+R or Cmd+Shift+R)
  3. Check browser console for errors
  4. Verify files are in S3 bucket
  5. Check CloudFront invalidation status

If cart operations fail:

  • Cart uses localStorage (API calls are disabled)
  • Check browser console for errors
  • Verify CartManager is initialized
  • Check that posts are loaded (allPosts array)
